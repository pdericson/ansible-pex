# ansible-pex

Ansible Python EXecutable

## Getting Started

```
curl -LO https://github.com/pdericson/ansible-pex/releases/download/0.1.0/ansible-2.4.2.0.pex
chmod +x ansible-2.4.2.0.pex
PEX_SCRIPT=ansible-playbook ./ansible-2.4.2.0.pex --version
```
